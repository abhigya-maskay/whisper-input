# dictation-app

Hyprland dictation helper via Faster Whisper.

## Installation

```bash
pip install -e .
```

## Usage

```bash
dictation-app --help
```

## Development

Install development dependencies:

```bash
pip install -e ".[dev]"
```

### Running tests

```bash
pytest
```

### Linting

```bash
ruff check .
```
